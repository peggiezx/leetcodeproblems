代码随想录训练营打卡 Day 10 | 232. Implement Queue using Stacks, 225. Implement Stack using Queues, 20. Valid Parentheses, 1047. Remove All Adjacent Duplicates In String

Posted on 2024-12-07 In Easy

打卡第10天！用栈并不局限于用Stack，而是要用Stack的思维，也就是如何操作最后进去的元素，可以用Array，StringBuilder，Deque等可以接触到最尾元素的结构来操作。

  1. Implement Queue using Stacks
  • two stacks
    • one for getting the input
    • one for producing the output
    • when we need to swap the order, pop everything to the output stack, once it’s done push them back to the input stack
  1. Implement Stack using Queues
  • two queues
    • the way to reverse the Queue: for every push, polling all the elements from q1 first and store it in q2 temporarily, and then add the lastest element to q1 first, then polling everything back to q1
    • q1[], q2[] => push(1)
      q1[1], q2[] => push(2)
      q1[], q2[1] ->
      q1[2], q2[1] ->
      q1[2, 1], q2[] =>push(3)
      q1[], q2[2, 1]
      q1[3], q2[2, 1]
      q1[3, 2, 1], q2[]
    • Queue, unlike Stack, will always keep the same order when you transfer one queue to another
    • The way to keep is reverse is to empty the queue and always push the lastest to an empty queue, this way you can keep the lastest push at the begining
  1. Valid Parentheses
  • push all the left parentheses
  • pop the right parenthese when:
    • stack is not empty && it matches the left parenthese
    • else return false
  • return if stack is empty
  1. Remove All Adjacent Duplicates In String
  • Intuition: use stack and StringBuilder => very slow (50%)
  • Optimized:
    • Only use StringBuilder
    • delete the last char of the sb using sb.deleteCharAt(len - 1)
