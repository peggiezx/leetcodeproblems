代码随想录训练营打卡 Day 3 | 203. Remove Linked List Elements, 207. Reverse the Linked list, 707. Design Linked List

Posted on 2024-11-29 Edited on 2024-12-04 In Easy

打开第三天！链表的基础复习，首先要明白链表的底层结构，特别是初始一个node时，明确是指向已有node还是创建一个新的node。另外是终止条件，避免NullPointerException。还有注意pointer的移动，是否需要保留现有值，哪个先移动，哪个后移动。

Linked list constructor

  1. Remove Linked List Elements
  • understand the underlying structure of linked list
    • when creating a new ListNode, e.g. dummy node, we are creating a new space
    • but if we simply assign a value to a ListNode, it creates a reference to the existing nodes, e.g. cur or pre, cur is a reference that points to where head points to; pre is a references that points to where dummy points to.
    • removing the nodes by cur and pre changes the nodes that head points to, so the dummy node updated too
  • when to update pre and when to update cur
    • pre used for skipping and always points to the new cur
    • cur used to check the val
    • NO update together: when cur.val == val, because the next cur is updated and pre already points to the new cur
    • YES update together: when cur.val != val, because cur moves forward, pre needs to move forward
  1. Reverse the Linked list
  • Intuitive: use a stack to store the value from the original list, and create a new list with the reversed values
  • Optimized: pre and cur
  1. Design Linked List
  • initialization: size, dummyHead
  • cur and pre values
  • use for loop instead of while to keep the code less error prone
