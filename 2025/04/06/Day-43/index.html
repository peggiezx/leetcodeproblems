  1. Max Subarray
    At each step, you decide:
  • Should I extend the current subarray with nums[i]
  • Or should I start fresh from nums[i]?
  1. is subsequence?

Solving with Edit Distance Perspective
This is a special case of Edit Distance, but with only deletion allowed on t.
We want to check if we can transform t into s by deleting some characters.

  • i → prefix of s
  • j → prefix of t

So dp[i][j] == true means:
We can transform the first j characters of t into the first i characters of s, using deletions only from t.

If current characters match, great — we “consume” both characters

Otherwise, we “delete” t[j-1] and try again
