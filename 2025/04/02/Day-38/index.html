Posted on 2025-04-02 Edited on 2025-04-07

Coin change类题目： 依赖两个维度：array里的硬币值，和target amount，但是只需要用一维的dp[amount]来记录
有时要在同一层里遍历所有硬币，或外层硬币、内层金额 → 两层循环。

  1. Coin Change
    让dp记录凑成金额x所需的最少硬币数
  2. 初始化:
  • 我们令 dp[0] = 0，表示“凑 0 不用硬币”。
  • 对 1..amount 的 dp[x] 全部填充一个大值（如 amount+1），表示暂时不可达。
  1. 双层循环：
  • 外层：x 从 1 到 amount，尝试每一个目标金额。
  • 内层：遍历所有硬币 coin。如果 coin <= x，则有机会用它来凑 x。
  • 更新 dp[x] = min(dp[x], dp[x - coin] + 1)。
  1. 输出结果：
  • 如果 dp[amount] 依然是初始的大值，返回 -1；否则返回 dp[amount]。

  1. Perfect Squares
    跟322 一样
    Time Complexity
    The outer loop runs n times.
    The inner loop (over i) runs up to sqrt(n) times.
    Overall complexity:
    O(n * sqrt(n))
