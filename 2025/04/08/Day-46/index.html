Day-46 | 动态规划背包问题12

Posted on 2025-04-08 Edited on 2025-04-16

回文串问题

  • 要加入字符串长度的维度len
  • 都可以用二维 dp[i][j] 解决
  • i 是子串的起点，j 是子串的终点，左下角到右上角遍历。j = i + len - 1; j < n

状态转移都涉及两端字符是否相等
s.charAt(i) == s.charAt(j) 是两个题的关键判断之一

  1. Palindrome Substring Count
    思路：
  • Go through all possible substrings of s, and for each one, check if it’s a palindrome.
  • if isPalindrome, count++

关键在于如何构建dp
dp[i][j] 表示s.substring(i, j + 1) 是否为palindrome

我们需要找到所有长度的substring, e.g. s=”abc”, 找到长度为1，并从所有index开始 => a, b, c; 长度为2 => ab, bc;

  1. Longest Palindromic Subsequence

dp[i][j] = the length of the longest palindromic subsequence between i and j

Base case:

Length 1: dp[i][i] = 1

Transition:

If s[i] == s[j]: they can be part of the palindrome → dp[i][j] = dp[i+1][j-1] + 2

Else: take the longer of skipping i or skipping j
