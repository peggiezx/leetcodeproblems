Day-52 | Island Problems

Posted on 2025-04-14 Edited on 2025-04-16

✅ 连通块 -> DFS/BFS 遍历
✅ 边界判断 -> 是否能出界、是否触到边
✅ 状态标记 -> 原地修改 or visited 数组
✅ 结果合并 -> 多个值传递 + 去重处理
✅ 模拟操作 -> 如翻转、涂色、改字符等

  1. Number of Closed Islands
    用DFS在寻找所有island的同时判断是否是closed
  • 如果island touch the boundary –> false
  • 如果island已经visited，或者是1 –> 还有潜力成为closed island –> true
  • return四个方向的合集
  1. Number of Enclaves
  • 用一个Pair class来统计
    • isEnclave
    • area
  • 如果该cell未被访问并且为1
    • dfs找到该岛屿的边界
    • 如果触到边界则isEnclave为false
    • 同时统计岛屿的大小
  • 只计算isEnclave岛屿的面积作为结果返回

    缺点：速度慢，没有必要计算所有的面积

优化解法

  • 找到边界开始所有的岛屿，dfs，将所有边界岛屿变成0
  • 统计非边界岛屿的数量
  1. Surrounded Region
    跟1020很像
  • 用visited记录比较好理解
  • 先从边界开始，如果找到O，那么dfs找到所有相连的O，并且标记为visited
  • 遍历非边界的cell，如果没有visited且为O，那么将此cell变成X
    优化
  • 可以用另一个字符来标记从边界开始的O形成的岛屿，就可以省去visited的空间
  1. Flood Fill
  • 用DFS从starting pixel开始搜索
  • 如果index越过边界停止搜索
  • 如果找到的cell不是starting pixel的value或者这个cell已经被找过了，停止搜索
  1. Making A Large Island
  • Two pass strategy
    • first pass: put all islands and its area in the map, key is a unique # and value is area
    • second pass: find all the 0s in the map; check four directions of 0; if there are any island, acculumate the area
    • return the max area;
